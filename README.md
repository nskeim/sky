Fully responsive.

## How to use
1. Login to your WonderCMS website.
2. Click "Settings" and click "Themes".
3. At the bottom of the "Themes" list paste this URL:
https://raw.githubusercontent.com/nskeim/sky-edited/master/wcms-modules.json
